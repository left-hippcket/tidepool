# 피시파더 ERP - 활어산지유통 웹 어드민

활어산지유통 ERP 시스템의 웹 어드민 목업 페이지입니다.

## 기술 스택

- **React 18** - UI 라이브러리
- **Vite** - 빌드 도구
- **Ant Design 5** - UI 컴포넌트 라이브러리
- **React Router DOM** - 라우팅

## 구현된 기능

### 1. 사업권역 관리 (`/territory`)
- **사업권역 탭**
  - 권역 목록 조회
  - 권역 추가/수정
  - 권역 비활성화 (소속 지역 체크)
- **상세지역 탭**
  - 지역 목록 조회 및 필터링
  - 지역 추가/수정
  - 지역 상태 변경

### 2. 셀러 관리 (`/seller`)
- 셀러 그룹 목록 조회
- 실시간 필터링 (검색어, 담당자, 품목분류, 사업권역, 상세지역, 상태)
- 정렬 기능 (매입액 누적/3개월/1개월, 최근거래일)
- 사업자등록증 체크박스 표시

### 3. 바이어 관리 (`/buyer`)
- 바이어 그룹 목록 조회
- 실시간 필터링 (검색어, 담당영업사원, 품목분류, 사업권역, 상세지역, 상태)
- 정렬 기능 (매출액 누적/3개월/1개월, 최근거래일)
- 사업자등록증 체크박스 표시

### 4. 상품 관리 (`/product`)
PRD v3 (3단계 독립 탭 구조) 기반 구현

#### 탭 1: 품목분류 관리
- 품목분류 목록 조회
- 품목분류 등록 (아이콘, 품목분류명)
- 품목분류 수정 (상태 변경, 비활성화 조건 체크)

#### 탭 2: 품목 관리
- 품목 목록 조회 및 품목분류별 필터링
- 품목 등록 (품목분류, 품목명, 주문단위, 주문단위당중량)
- 품목 수정 (비활성화 조건: 소속 원산지/규격 체크)
- 원산지/규격 개수 표시

#### 탭 3: 원산지/규격 관리
- **3열 레이아웃**
  - 왼쪽: 품목 선택 트리
  - 중앙: 선택 품목의 원산지 카드 배열
  - 우측: 선택 품목의 규격 카드 배열
- 원산지/규격 독립 추가/수정
- 원산지/규격 비활성화 (독립적, 조건 없음)

## 목업 데이터

`src/data/mockData.js`에 다음 데이터가 포함되어 있습니다:

- **사업권역**: 7개 권역 (수도권, 강원권, 경상권, 충청권, 호남권, 제주권, 조인유통)
- **상세지역**: 24개 지역
- **품목분류**: 8개 분류 (누운고기, 뜬고기, 갑각류, 연체류, 패류, 건어물, 굴류, 횟감)
- **품목**: 10개 품목 (광어, 넙치, 우럭, 강도다리, 도미, 농어 등)
- **원산지**: 8개 원산지
- **규격**: 8개 규격
- **셀러 그룹**: 10개 그룹 (총 매입액, 최근 3개월/1개월 매입액, 최근거래일 포함)
- **바이어 그룹**: 10개 그룹 (총 매출액, 최근 3개월/1개월 매출액, 최근거래일 포함)
- **담당자**: 최용환, 이시호, 노원진, 고영석, 박현재

## 주요 기능 시연

### 1. 사업권역 관리
- 권역 추가 시 자동으로 표시순서 설정
- 권역 비활성화 시 소속 활성 지역 체크
- 상세지역 필터 버튼으로 권역별 필터링

### 2. 셀러/바이어 관리
- 실시간 검색 및 필터링 (검색 버튼 불필요)
- 테이블 컬럼 헤더 클릭으로 정렬
- 매입액/매출액 억 단위 표시

### 3. 상품 관리
- 3단계 독립 탭 구조로 관리 복잡도 감소
- 품목분류 → 품목 → 원산지/규격 순차적 관리
- 비활성화 조건 체크 (하위 항목 활성 여부)
- 원산지/규격 독립 관리 (상호 의존성 없음)

## 구현 특징

- **AntD 컴포넌트 활용**: Layout, Menu, Table, Modal, Form, Select 등
- **실시간 필터링**: 입력 즉시 테이블 업데이트
- **정렬 기능**: 컬럼 헤더 클릭으로 오름차순/내림차순 전환
- **모달 기반 CRUD**: 등록/수정 시 모달 폼 사용
- **유효성 검사**: 필수 입력, 중복 체크, 형식 검증
- **비활성화 조건**: 하위 항목 체크 후 비활성화 허용
- **반응형 레이아웃**: Sider 접기/펼치기 기능
- **상태 관리**: React Hooks (useState) 사용
